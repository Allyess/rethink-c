Detailed Description

Big number operation: Addition, Subtraction, Multiplication, and Division. Use char string to store big number.

Bitmap to store 0/1 values.

Author
hutusi (hutus.nosp@m.i@ou.nosp@m.tlook.nosp@m..com)

Before arithmetic calculations, use bignum_int_sanitize to sanitize the char string.

To do addition, use bignum_int_addition.

To do subtraction, use bignum_int_subtraction.

To do multiplication, use bignum_int_multiplication.

To do division, use bignum_int_division.

◆ bignum_int_sanitize()

Sign bignum_int_sanitize ( char *  num)

Sanitize the number(char string).

For the big number is represented as char string, it should be sanitized before do calculations. e.g., "0001 " "0001" "-123456" will delete 0s or blanks, and the sign '+' or '-' will be deleted too, return the sign.

Parameters
numThe number be both input and output para. If the input number is a invalid number string (like "ABC001"), it will be output as "0".
Returns
Sign The number's sign.

Sanitize the number(char string).

examples: "0000000" => "0" ret Positive "0000123400" => "123400" ret Positive " +0000123400 " => "123400" ret Positive "-0000000" => "0" ret Negative "-0000123400" => "123400" ret Negative " -0000123400 " => "123400" ret Negative " A-0000123400 " => "0" ret NaN
