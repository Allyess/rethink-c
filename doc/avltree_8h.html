Detailed Description

AVL Tree.

Author
hutusi (hutus.nosp@m.i@ou.nosp@m.tlook.nosp@m..com)

AVL tree is a self balancing binary search tree, where difference of right subtree and left subtree height to a node is at most 1.

Georgy Adelson-Velsky and Evgenii Landis introduced it in 1962.

Date
2019-07-30
